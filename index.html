AI Assistant - FRIDAY

This project is an AI assistant named FRIDAY, created using Python. It can perform various tasks such as opening websites, playing music, telling jokes, and more.


import datetime
import os
import speech_recognition as sr
import pyttsx3
import webbrowser
import subprocess

def change_voice(engine, voice_id):
    voices = engine.getProperty('voices')
    engine.setProperty('voice', voices[voice_id].id)

def say(text, voice_id=1):
    engine = pyttsx3.init()
    change_voice(engine, voice_id)
    engine.say(text)
    engine.runAndWait()

def takeCommand():
    r = sr.Recognizer()
    with sr.Microphone() as source:
        r.pause_threshold = 1
        try:
            print("Listening...")
            audio = r.listen(source, timeout=5)
            print("Recognizing...")

            query = r.recognize_google(audio, language="en-in")
            print(f"User said: {query}")
            return query.lower()

        except sr.UnknownValueError:
            print("Sorry, I did not hear anything.")
            return ""
        except sr.RequestError as e:
            print(f"Could not request results from Google Web Speech API; {e}")
            return "Some error occurred. Sorry."

if __name__ == '__main__':

Car Racing Game

This project is a car racing game developed in Unity. The game features realistic physics, various tracks, and an immersive gaming experience.

Features:

  • Realistic car physics
  • Multiple tracks
  • Dynamic weather conditions
  • Multiplayer mode

Below is a snippet of the core code for the car controls:


// Unity C# Script for Car Control
using UnityEngine;

public class CarController : MonoBehaviour
{
    public float speed = 10.0f;
    public float turnSpeed = 50.0f;

    void Update()
    {
        float moveDirection = Input.GetAxis("Vertical") * speed * Time.deltaTime;
        float turnDirection = Input.GetAxis("Horizontal") * turnSpeed * Time.deltaTime;

        transform.Translate(0, 0, moveDirection);
        transform.Rotate(0, turnDirection, 0);
    }
}
